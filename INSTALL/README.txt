
==================================
OPENMUSIC INSTALLATION
==================================

- COPY THE OM 6.x DIRECTORY ANYWHERE
(STANDARD INSTALL LOCATION IS YOUR SYSTEM'S "/Applications/" FOLDER)

- FOR A FIRST OPENMUSIC INSTALLATION THE FONTS IN THE "Fonts" FOLDER WILL NEED TO BE INSTALLED.
=> DOUBLE-CLICK THE .ttf FILES AND INSTALL USING MACOS FONT INSTALLATION UTILITIES
